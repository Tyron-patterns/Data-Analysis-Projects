# 🚲 Bike Rental Analysis (Power BI)

## 📌 Overview
This project analyzes bike rental data using Power BI to uncover demand patterns, seasonality effects, and operational insights. The objective is to transform raw rental records into an interactive dashboard that supports data-driven decision-making.

## 📊 Dataset
The dataset includes rental counts along with contextual variables such as:

- Date and time  
- Season and weather conditions  
- Temperature and humidity  
- Working day / holiday indicators  

## 🛠 Data Preparation & Modeling

- Cleaned and validated data using Power Query  
- Built a star schema with a dedicated Date dimension  
- Created relationships between fact and dimension tables  
- Developed DAX measures for performance tracking  

## 📈 Key Insights

- Clear seasonality patterns with peak demand in warmer periods  
- Weather conditions significantly influence rental volume  
- Noticeable differences between working days and holidays  
- Time-based trends useful for forecasting and resource planning  

## 💡 Skills Demonstrated

- Data cleaning and transformation  
- Dimensional modeling  
- DAX and time-intelligence calculations  
- Dashboard design and business-oriented storytelling

# ⚡ ElectroHub Sales Analysis (Power BI)

## 📌 Overview
This project analyzes sales data for ElectroHub, an electronics retail company, using Power BI. The objective is to evaluate revenue performance, customer behavior, and product-level trends to support strategic business decisions.

The dashboard provides interactive insights into sales growth, profitability, and regional performance.

---

## 📊 Dataset
The dataset includes transactional sales records with variables such as:

- Order date  
- Product category and subcategory  
- Sales revenue  
- Quantity sold  
- Customer segment  
- Regional information  

---

## 🛠 Data Preparation & Modeling

- Cleaned and transformed data using Power Query  
- Handled missing values and standardized categorical fields  
- Built a star schema data model  
- Created calculated columns and DAX measures for KPI tracking  
- Implemented time-intelligence functions for trend analysis  

---

## 📈 Key Insights

- Identified top-performing product categories and regions  
- Detected seasonal sales patterns  
- Analyzed customer segment contribution to revenue  
- Evaluated profitability and growth trends over time  

---

## 💡 Skills Demonstrated

- Data cleaning and transformation  
- Dimensional modeling  
- DAX calculations and time-intelligence  
- Business KPI development  
- Interactive dashboard design and storytelling  

---

## 🖥 Tools Used

- Power BI Desktop  
- DAX  
- Power Query
