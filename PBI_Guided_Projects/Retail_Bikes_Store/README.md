# 🚲 Retail Bike Store — Power BI Project

## TL;DR
End-to-end **Power BI dashboard** for a global retail bike store, built as part of a **guided Power BI project**.  
Focus on **data modeling, DAX measures, interactive dashboards, and business-oriented insights**.

---

## 📌 Project Overview
This project analyzes sales, profitability, customer behavior, and returns for a global retail bike store.  
The goal was to design an **interactive, decision-support dashboard** that enables stakeholders to explore performance across time, geography, products, and customers.

The project was completed as a **guided Power BI project** as part of an online Power BI course by **Maven Analytics**, with full hands-on implementation.

---

## 🛠 Tools & Technologies
- Power BI Desktop  
- DAX (measures, KPIs, time intelligence)  
- Data Modeling (star schema)  
- Interactive visuals & slicers  

---

## 📊 Key Features
- Executive KPIs: **Revenue, Profit, Orders, Return Rate**
- Time-series analysis with trend lines and drill-down
- Product and category performance analysis
- Customer segmentation and top-customer analysis
- Geographic analysis (regions & countries)
- Target vs actual performance indicators
- What-if analysis using **price adjustment parameters**

---

## 🧩 Data Model
- Star schema with dedicated **fact and dimension tables**
- Separate lookup tables for:
  - Date
  - Customers
  - Products & Categories
  - Geography / Territory
- Dedicated **measure table** for cleaner DAX organization
- Disconnected tables for:
  - Metric selection
  - Price adjustment scenarios

---

## 🎯 Learning Outcomes
- Translating business questions into analytical dashboards
- Structuring a scalable Power BI data model
- Writing reusable and readable DAX measures
- Designing dashboards for **clarity and decision-making**
- Applying best practices for guided but realistic business scenarios

---

## ℹ️ Notes
- This is a **guided project**, completed as part of a Power BI learning program.
- All modeling, measures, and visuals were implemented hands-on.
- The project is intended to demonstrate **Power BI proficiency**, not proprietary business insights.
