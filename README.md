# 📊 Data Analysis Projects

A centralized portfolio collecting multiple data analysis projects developed across **SQL, Python, Excel, and Power BI**.

Each project is self-contained and documents an end-to-end analytical workflow, from raw data to structured insights and reporting.

---

## TL;DR

**Raw data → Excel / SQL → Python & SQL analysis → Power BI modeling → Insights & reporting**

This repository acts as a **portfolio hub**. Each subfolder is a standalone data analysis project with its own documentation.

---

## 🔎 Overview

Most projects in this repository follow a common analytical pipeline:

1. **Data collection & initial structuring**
   - Raw data compiled or collected in **Excel**
   - Early exploration and validation in Excel and/or SQL

2. **Data analysis & transformation**
   - Data cleaning, aggregation, and feature engineering using **SQL** and **Python**
   - Exploratory Data Analysis (EDA) to study distributions, trends, and anomalies

3. **Formalization & integration**
   - Structured data models
   - Integration into **Power BI** for scalable analysis and visualization

4. **Insights & reporting**
   - Analytical findings documented via dashboards, notebooks, SQL scripts, and written summaries
